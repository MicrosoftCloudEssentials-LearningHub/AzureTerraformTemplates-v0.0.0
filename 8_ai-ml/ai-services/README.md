> This template creates a Cognitive Services account (Azure AI Services). Configure `kind` for your workload (for example OpenAI, SpeechServices, TextAnalytics).

## Usage

```sh
az login
terraform init -upgrade
terraform validate
terraform plan
terraform apply -auto-approve
```
